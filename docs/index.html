nzilbb.elpis Documentation

This is client library for communicating with Elpis web application servers.

What is Elpis?

Elpis is a tool to obtain a first-pass automatic transcription on untranscribed audio, for manual correction.

Elpis was developed by the Centre of Excellence for the Dynamics of Language (CoEDL) as part of the Transcription Acceleration Project (TAP) cross-disciplinary project aimed at identifying the workflows of linguists and language workers during transcription, and developing assistive tools to accelerate that process.

It has a user-friendly browser-based interface, which can be used by researchers to upload training data, initiate model training, and provide recordings for automatic transcription.

This GUI uses an API which is documented here.

What is this library?

The library provides programmatic access to Elpis API for Java, developed by the New Zealand Institute of Language, Brain and Behaviour (NZILBB).

Detailed JavaDoc object model documentation is available here.
